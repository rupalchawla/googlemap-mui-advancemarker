# Google AdvanceMarker With MUI

[![GNU License](https://img.shields.io/badge/GNU_General-License-blue)](https://github.com/rupalchawla/googlemap-mui-advancemarker/blob/main/LICENSE)

This is a TypeScript / React Repo that uses [`@vis.gl/react-google-maps`] and [Material React UI](https://mui.com/)

## Installation

Update `config.js` with `API_KEY` and `MAP_ID`

```sh
npm install
npm start
```
